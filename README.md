### 第一次上傳到github:
1. github new repository 
2. git clone 連結
3. git add .
4. git commit -m "改什麼東西內容"
5. git push把改好東西上傳github

## git指令 : 
* git pull先把最新版本下載下來
* git status: 現在狀態
* git diff:可以看到上次改了什麼
